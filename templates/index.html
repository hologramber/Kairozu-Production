Kairozu uses programmed learning to build an understanding of Japanese grammar using English to Japanese sentence construction. Most sentences are computer-generated using common words and grammar patterns compiled from Japanese language sources aimed at beginners.

Learn grammar and relevant vocabulary in small, logical steps. Sentence comprehension increases as new grammar patterns mix with those you already know. Spaced-repetition review helps retain what you've learned; avoid the visual recognition trap that usually occurs when using flashcards.
