Kairozu uses programmed learning to build an understanding of Japanese
grammar using English to Japanese sentence construction.

Learn grammar and relevant vocabulary in small, logical steps.
Less focus on reading about grammar and more focus on using grammar in context.

Practice grammar recall by translating English sentences into Japanese.
Sentence comprehension increases as new grammar patterns mix with those you already know.

Most sentences are computer-generated using common words and grammar
patterns compiled from Japanese-language sources aimed at beginners.

Review vocabulary and grammar via spaced-repetition to practice recognition;
avoid the visual memorization-recognition trap that usually occurs when using flashcards.
